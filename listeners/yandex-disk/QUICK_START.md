# Быстрый старт - получение токена и тестирование

Если у вас уже есть **Client ID** и **Client Secret**, следуйте этой инструкции.

## Простой способ (рекомендуется)

### Шаг 1: Получите токен через браузер

1. Откройте в браузере этот URL (замените `YOUR_CLIENT_ID` на ваш Client ID):

```
https://oauth.yandex.ru/authorize?response_type=token&client_id=YOUR_CLIENT_ID
```

2. Войдите в Яндекс и разрешите доступ
3. После перенаправления в URL будет токен: `access_token=ВАШ_ТОКЕН`
4. Скопируйте токен

**Готово!** Теперь у вас есть токен для тестирования.

### Шаг 2: Проверьте токен

```bash
deno task health ВАШ_ТОКЕН
```

## Альтернативный способ (через код авторизации)

Если первый способ не работает, используйте скрипт:

```bash
deno run --allow-net --allow-env scripts/get-token-full.ts YOUR_CLIENT_ID YOUR_CLIENT_SECRET
```

Скрипт:
1. Откроет браузер для авторизации
2. Попросит ввести код авторизации из URL
3. Автоматически обменяет код на токен
4. Проверит токен

## Что дальше?

После получения токена:

1. **Проверьте работоспособность:**
   ```bash
   deno task health ВАШ_ТОКЕН
   ```

2. **Запустите тесты:**
   ```bash
   deno task test
   ```

3. **Проверьте настройку:**
   ```bash
   deno task verify
   ```

## Хранение токена

Сохраните токен в переменной окружения:

```bash
# Windows (PowerShell)
$env:YANDEX_DISK_TOKEN="ваш_токен"

# Linux/Mac
export YANDEX_DISK_TOKEN="ваш_токен"
```

Теперь можно использовать:
```bash
deno task health
```

**Важно:** Не коммитьте токен в git!
